顺便把地形的瓦片贴图做了一下，边框好像太粗了点，之后再改。

```markdown
地图编辑器用户行为清单
一、鼠标操作（在主画布上）
操作	方式	行为
单格绘制	笔刷模式下，左键单击六边形	将当前选中属性写入该瓦片
连续绘制	笔刷模式下，左键按住拖拽	连续所经瓦片全部写入属性
笔刷半径覆盖绘制	设置笔刷半径 ≥1 环，左键单击	以点击瓦片为中心，BFS 展开 N 环，批量写入属性（1 次撤销）
笔刷半径拖拽绘制	笔刷半径 ≥1 环，左键按住拖拽	拖拽路径上的每个瓦片均展开半径覆盖
单选瓦片	框选模式下，左键单击六边形	切换该瓦片的选中状态（加入/移出选中集）
矩形框选	框选模式下，左键拖拽矩形	框内所有六边形加入选中集
批量编辑选中瓦片	框选一批瓦片 → 切回笔刷 → 点击任一中选瓦片	当前属性批量写入全部选中瓦片（1 次撤销）
清除过期选中	有旧选中时，笔刷模式点击选中区外的瓦片	自动清除选中集，正常笔刷
悬停预览	鼠标在画布上移动	悬停瓦片红色高亮边框；笔刷半径 ≥1 且无选中时，显示金色虚线半径覆盖预览
平移视图	中键按住拖拽	视口平移，鼠标拖多少像素视图滚多少（与缩放级别无关）
缩放视图	滚轮上/下	以鼠标所在瓦片为中心放大/缩小（0.15×～4×）
二、键盘快捷键
快捷键	行为
Ctrl+N	新建地图
Ctrl+O	打开地图（JSON 文件）
Ctrl+S	保存地图
Ctrl+Shift+S	另存为...
Ctrl+Z	撤销（最近一次单格/批量编辑）
Ctrl+Y / Ctrl+Shift+Z	重做
Ctrl++	放大视图
Ctrl+-	缩小视图
Ctrl+0	重置视图（缩放 100%，回到原点）
Ctrl+M	切换小地图窗口显示/隐藏
Ctrl+Q	退出编辑器
Esc	清除所有选中瓦片
三、左侧面板 — 控件操作
面板	操作	行为
小地图按钮	点击「🗺 小地图」按钮	切换小地图独立窗口显示/隐藏
工具条	点击「🖌 笔刷」	切换到笔刷模式
点击「⬜ 框选」	切换到框选模式
调节「尺寸」数值框 (0~5 环)	设置笔刷半径：0=单格，1=7格，2=19格，3=37格，4=61格，5=91格
图层面板	点击图层按钮（地形/群落/构造/资源）	切换当前编辑图层，右侧属性面板联动刷新
勾选/取消图层复选框	切换该图层在主画布上的可见性
属性面板	点击属性值按钮（如「01 平原」「14 村庄」）	选中笔刷属性值
点击「✕ 擦除」	笔刷值设为空（擦除当前图层属性）
四、菜单栏操作
菜单	操作	行为
文件 → 新建	点击 / Ctrl+N	弹出新建对话框，输入名称和行列数 (5~200)
文件 → 打开...	点击 / Ctrl+O	弹出文件选择器，加载 JSON 地图
文件 → 保存	点击 / Ctrl+S	保存当前地图为 JSON
文件 → 另存为...	点击 / Ctrl+Shift+S	选择新路径保存 JSON
文件 → 退出	点击 / Ctrl+Q	若未保存弹出确认对话框，随后关闭
编辑 → 撤销	点击 / Ctrl+Z	撤销
编辑 → 重做	点击 / Ctrl+Y	重做
视图 → 放大	点击 / Ctrl++	放大视图
视图 → 缩小	点击 / Ctrl+-	缩小视图
视图 → 重置视图	点击 / Ctrl+0	重置
视图 → 小地图	点击 / Ctrl+M	切换小地图窗口
五、小地图窗口操作
操作	行为
点击瓦片	主画布视口自动跳转到该瓦片为中心
拖拽窗口	自由移动小地图窗口位置
关闭窗口（点 X）	隐藏小地图，同步取消菜单/工具栏勾选
被动更新	主画布缩放/平移/编辑/悬停时，小地图实时刷新视口矩形和焦点圆环
实时缩略图	始终显示全地图的微型六边形渲染（地形底色 + 群落叠加 + 构造边框 + 资源圆点）
六、对话框操作
对话框	操作	行为
新建地图	输入地图名称	自定义地图显示名
调节行数 / 列数 (5~200)	设定六边形网格规模，默认 30×40
点击「OK」	创建空白地图并加载到画布
保存地图	选择路径 + 文件名	导出为 JSON 文件
打开地图	选择 JSON 文件	加载已有地图
未保存确认	点击「保存并退出」	触发保存流程后关闭
点击「不保存直接退出」	丢弃修改直接关闭
点击「取消」	返回编辑器
七、状态栏信息（实时只读）
显示内容	更新时机
悬停瓦片坐标 (row, col) tile #id	鼠标移动
当前编辑图层（地形/群落/构造/资源）	图层切换
当前工具模式 + 笔刷半径 工具: 笔刷 (7格)	工具/半径切换
当前笔刷属性值 笔刷: 01	属性选择/擦除
八、完整编辑工作流示例

打开编辑器 → 新建 30×40 地图 → 画布显示空六边形网格
  → 左侧选「地形」图层 → 点击「01 平原」
  → 画布左键单击铺底色
  → 调笔刷半径 2 环 → 大面积刷山脉/河流
  → 切「群落」图层 → 选「05 森林」→ 在地形上叠加
  → 切「构造」图层 → 选「14 村庄」→ 放置聚落
  → 切「资源」图层 → 选「02 煤矿」→ 标记资源点
  → Ctrl+Z 撤销不满意的操作
  → 滚轮缩放 / 中键平移调整视角
  → 打开小地图 Ctrl+M → 点击跳转
  → Ctrl+S 保存为 JSON
```
